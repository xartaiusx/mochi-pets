# Mochi Pets Docs

Mochi Pets is a closed Mochirii playtest where approved members enter one shared guild room, create a curated character, meet Lirabao, and care for the guild pet together. All playtest progress has no real value.

## Player-Facing Playtest Docs

- [Alpha Preview](alpha-preview-ready.md): what members can do, who can enter, what is not included, and what must work before the live tester page is ready.
- [Unity Bootstrap](unity-bootstrap.md): how the current Unity WebGL room is opened, checked, and built locally.
- [Website Integration](site-integration.md): the stable routes and browser messages used by the Mochirii website doorway.

## Maintainer Notes

The other files in this folder are maintainer notes for local checks, safety boundaries, future planning, release evidence, and deployment approval prep. They are not player copy and should not be shown on the tester page or in public game UI.

Older notes that mention the previous browser runtime, future asset-provider work, or funding plans are archive/future-planning material only. The current playtest remains one Unity WebGL room, three curated character presets, one shared Lirabao, member sign-in for saved play, and no real value.
