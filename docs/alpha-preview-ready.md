# Mochi Pets Alpha Preview

Mochi Pets Alpha Preview is the closed guild playtest for the shared Mochirii room.

Approved members enter one 3D room, create a curated character, meet Lirabao, care for the guild pet together, leave, return, and see saved progress.

## What Members Can Do

- Enter the shared guild room from the Mochirii tester page.
- Create a character from three curated presets.
- Move around the Jade Lantern room in a desktop browser.
- Wave or send a simple room signal.
- Meet Lirabao and care for the guild pet together.
- Return later with saved character and Lirabao progress.

## Entry Rules

- The tester password opens the playtest page.
- Saved play requires Mochirii member sign-in.
- Only approved testers can enter the saved playtest.
- Testers must accept the playtest terms before joining the room.

## Playtest Boundaries

- All progress has no real value.
- Avatar uploads are not part of this playtest.
- There is only one shared room.
- There is only one shared starter pet.
- Desktop browser play is the current target.
- Paid item value, real-money features, multiple rooms, and mobile-specific play are not part of this playtest.

## Ready For The Live Tester Page

The alpha is ready for the live tester page when:

- The tester password wall is active.
- Member sign-in is required for saved play.
- The shared room loads instead of an old fallback.
- Signed-out visitors, non-testers, and terms-missing testers are blocked.
- A valid tester can create a curated character.
- Character progress returns after reload and sign-in.
- Two testers can see distinct characters in the same room.
- Both testers can care for the same Lirabao state.
- Lirabao progress returns after reload and sign-in.
- Invalid character preset IDs are rejected.
- The page remains hidden from search indexing.
- Public copy says closed playtest and no real value.

## If The Room Is Unavailable

The tester page should stay online. The game frame should show a friendly playtest paused message, and the old room should not silently open.
