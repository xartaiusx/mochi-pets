# Asset Ledger

All current art is project-authored crisp Cozy Wushu pixel-kit art generated by `apps/game/scripts/generate-assets.mjs` from HD source-export intent. Kenney's free asset catalog is used only as an art-direction reference for compact readability, kit cohesion, and small-scale silhouettes; no Kenney files are copied, traced, downloaded into runtime, vendored, or required at runtime. No third-party art, marketplace packs, AI image downloads, or unverified assets are used.

Generation prompt/source note for this set: "High-fidelity source intent for a crisp Cozy Wushu 2D RPG asset for Mochi Social, exported as a small readable pixel-kit runtime asset, top-down 2D RPG clarity, Kenney-informed compact silhouette clarity, dusk lantern lighting, jade green and red timber palette, warm gold rim light, soft contact shadow, clear silhouette at gameplay zoom, no text, no logos, no real-world brands, alpha preview no-real-value game asset." The assets are deterministic code-generated PNGs and ship under the repository MIT license. Source-art prompt notes live in `assets/source/game/prompts.md`; HD source-export intent lives in `assets/source/game/hd-source-export.md`; reference analysis lives in `docs/kenney-reference-analysis.md`; implementation rules live in `docs/game-art-bible.md` and `docs/asset-pipeline-contract.md`.
