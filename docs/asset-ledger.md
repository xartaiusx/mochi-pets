# Asset Ledger

All current runtime art is original Mochirii High-Fidelity Wuxia art generated for this project by `apps/game/scripts/generate-assets.mjs`. The source-export lane is documented in `assets/source/game/hd-source-export.md`; the pipeline creates source-master PNGs and source cards under `assets/source/game/hd/`, then exports runtime PNGs with Sharp. No third-party art, marketplace packs, screenshots, copied names, or external-project visual derivatives are used.

Generation note: smooth illustrated 2D Mochirii guild/wuxia art with soft lantern lighting, jade, lacquer, gold, silk, and paper materials, clear gameplay silhouettes, no baked-in text, no logos, no real-value implication, and Enjin Canary kept as a configured preview stub. Assets ship under the repository MIT license as project-authored/generated-for-project work.
